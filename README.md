# Setup LimsPlus Offline Mode

## Step 1

- Download docker desktop app and install system
- https://www.docker.com/products/docker-desktop/

## Step 2

### Clone Repo

- git clone https://github.com/appasaheb4/limsplus-offline.git
- cd limsplus-offline
- docker-compose up

## Step 3

### Open Browser

- http://localhost:3000
